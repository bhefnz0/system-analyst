# system-analyst

## Анализ требований (Функционал корзины)
**Логические противоречия и недочеты:**
1. макс. число товаров (пункт 3, пункт 4)
  1.1 Проблема
    - в этих пунктах говорится об одном и том же, так как "различные товары" - это конкретика, а "все товары" - это обобщение ("все товары" подразумевают в себе различность), под которое попадает первый пример (в пункте 3). 
  1.2 Решение:
    - мы должны объединить логику под один пункт:
      - Пользователь может изменять количество каждого товара в корзине.
      - Для явного удаления товара предусмотрена отдельная кнопка (крестик).
      - Если пользователь через счетчик уменьшает количество товара до 0, этот товар также автоматически удаляется из корзины.
    
2. удаление (пункт 2, пункт 9)
  2.1 Проблема:
    - оно заключается в том, что пункт 2 говорит нам, что для удаления товара из корзины должна использоваться ТОЛЬКО отдельная кнопка, но в это же время пункт 9 указывает на возможость изменения количество товара для его удаления.
  2.2 Решение:
    - в корзине действуют строгие лимиты: не более 5 уникальных позиций, при этом суммарное количество всех единиц товаров (с учетом их количества) не должно превышать 20 штук. При попытке превысить эти лимиты система должна блокировать добавление и показывать пользователю соответствующее уведомление.
    
3. цена (пункт 7, пункт 13)
  3.1 Проблема:
    - Неполнота требований при пересечении условий друг с другом.
  3.2 Решение:
    - Цена на товар в корзине синхронизируется с каталогом. Если стоимость товара изменилась в базе данных (то есть в каталоге), система автоматически обновляет ее в корзинах всех пользователей. Фиксация цены должна происходить строго в момент создания заказа.

## Проектирование API
**REST API запрос при переходе в меню выбора магазина**
Javascript/Typescript:
```
fetch('https://m.petryshka-green.ru/api/markets')
  .then(response => {
    const allMarkets = response.json();
    console.log(allMarkets);
    // Вывожу список всех магазинов в консоль
  })
  .catch(error => console.error('Ошибка:', error));
```
